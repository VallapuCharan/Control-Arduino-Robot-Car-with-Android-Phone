# Control-Arduino-drone-with-Android-Phone
Full control of Arduino drone with Android phone using bluetooth.

1. Drone is developed using Arduino chip, bluetooth device, ultra-sonic sensors and led lights and andriod phone.
2. Using this code one can completely control the drone
 Drone can perform following actions.
 -> It can move Forward and backward.
 -> It can move right and left.
 -> It can turn on its head lights.
 -> It can turn on its parking lights.
 -> It can detect obstacles using ultra sonic sensors.
